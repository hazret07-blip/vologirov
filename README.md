Описание приложения «Book Tracker» Автор: ШВологиров Аслан

Название: Book Tracker (Трекер прочитанных книг)

Цель проекта: создать удобное GUI‑приложение для учёта прочитанных книг с возможностью фильтрации, сохранения данных в формате JSON и версионного контроля через Git.

Функциональность Приложение позволяет:

Добавлять книги с указанием:

названия;

автора;

жанра;

года издания;

количества страниц;

рейтинга (по 10‑балльной шкале);

статуса прочтения («Прочитано», «В процессе», «Планирую прочитать»).

Отображать список книг в виде интерактивной таблицы с возможностью сортировки по любым колонкам.

Фильтровать книги по:

жанру (выпадающий список с уникальными жанрами);

году издания (диапазон или конкретный год);

статусу прочтения;

рейтингу (выше/ниже указанного значения).

Редактировать и удалять записи — через контекстное меню или кнопки в интерфейсе.

Сохранять и загружать данные в формате JSON:

автоматическое создание файла books.json при первом сохранении;

поддержка перезаписи и дополнения существующих данных;

валидация структуры JSON при загрузке.

Проверять корректность ввода:

обязательные поля (название, автор);

год издания — целое число в диапазоне 1000–2100;

количество страниц — положительное целое число;

рейтинг — число от 0 до 10 с поддержкой дробных значений.

Интегрироваться с Git для версионного контроля:

инициализация локального репозитория;

настройка .gitignore для исключения временных файлов;

коммиты при сохранении данных (опционально).

Технические требования Язык программирования: Python 3.8+

GUI‑библиотека: tkinter (встроенная) или PyQt5 (для расширенного интерфейса)

Хранение данных: JSON (json модуль Python)

Система контроля версий: Git

Зависимости: отсутствуют (для базового варианта)

Инструкция по запуску Установите Python 3.8 или новее.
